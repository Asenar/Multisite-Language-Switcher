    has_value

    has_value(string $key) : bool

    Check if the array has an non empty item with the specified key. This is method is similar to the overloaded __isset-method since __set cleans empty properties but I use for example $obj->has_value( $temp ) and not isset( $obj->$temp ) which is the same but a little bit ugly.

    $obj = new MslsGetSet;
$val = $obj->has_value( 'temp' ); // false == $val
$obj->temp = 'test';
$val = $obj->has_value( 'temp' ); // true == $val
